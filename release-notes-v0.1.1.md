# 연아의 수학 모험 v0.1.1

실제 Android 기기에서 화면의 한국어 글자가 사라지고 개발 콘솔에 예외가 반복되던
`v0.1.0` 문제를 수정한 테스트 버전입니다.

- Pretendard TMP 글꼴, 머티리얼, 432개 글리프 아틀라스를 빌드 전에 생성·검증
- Android 실행 중 글꼴을 즉석 생성하던 실패 경로 제거
- 배포 APK에서 Unity 개발 빌드, 플레이어 디버깅, 개발 콘솔 비활성화
- 기존 3개 미니게임, 초기 실력 확인, 적응형 난이도, 로컬 학습 기록 유지
- 같은 패키지(`com.yeona.mathadventure`)와 서명, 높아진 버전 코드로 기존 앱 위에 업데이트 가능

검증 결과: Unity BuildReport 성공(오류 0), EditMode 테스트 81/81 통과,
Android 7.1(API 25) 이상, ARMv7/ARM64 지원.
